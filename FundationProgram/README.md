# Fundation Program in Java

Se requiere un programa que permita solucionar el problema que tiene una fundacion de ayudas
comunitarias, esta fundación pierde tiempo registrando diariamente a sus empleados,
colaboradores y beneficiarios, se entiende que un empleado se diferencia del resto por
tener salario, cargo y horario (8-5 pm) o (2-11 pm), un colaborador solo se diferencia por
tener horas al mes, una meta que se pone de forma voluntaria y recibe una bonificacion a
fin de mes por eso, (cada hora se paga a 55 mil pesos y puede tener maxima 25) y un
beneficiario tiene como campos propios, cantidad de ayudas por mes, solo puede tener 5,
recordar que todos son personas que a su vez tienen documento, nombre, apellido, edad, direccion.
Ahora bien, se espera que, a los 3 se le implementen 1 misma función, que es "presentarse" y por supuesto
una clase abstracta en "funciones" que permita "registrar persona, y ver personas".
Si desea añadir una interface es opcional.
